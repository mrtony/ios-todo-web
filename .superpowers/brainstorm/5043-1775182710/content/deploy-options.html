免費部署平台比較

以下都有免費方案，適合個人專案。選擇你偏好的部署方式

A

Render

免費方案支援 Node.js 服務 + 靜態網站

  • 連結 GitHub repo 自動部署
  • 免費方案：閒置 15 分鐘後休眠，首次請求需冷啟動 (~30s)
  • 支援持久化磁碟（SQLite 友善）
  • 自訂域名、HTTPS 自動設定

⭐ 推薦 — 對 Express + SQLite 最友善

B

Railway

每月 $5 免費額度（約 500 小時執行時間）

  • 連結 GitHub repo 自動部署
  • 不會休眠，回應速度快
  • 支援持久化磁碟（付費方案）
  • 免費額度用完需升級，SQLite 資料可能不持久
C

Fly.io

免費方案提供 3 個共享 CPU VM

  • 透過 Dockerfile 部署，支援全球多區域
  • 內建持久化 Volume（SQLite 完美搭配）
  • 需要安裝 flyctl CLI
  • 設定稍微複雜一點，但效能好
D

Vercel + GitHub Pages 混合

前端 GitHub Pages + 後端 Vercel Serverless Functions

  • 前端真正部署在 GitHub 上
  • Vercel 免費方案很慷慨（每月 100GB 頻寬）
  • ⚠️ Serverless 環境不支援 SQLite 檔案持久化
  • 需改用 Turso (SQLite 雲端) 或其他遠端 DB
E

Zeabur

亞洲開發者友善的部署平台，有免費額度

  • 連結 GitHub repo 一鍵部署
  • 介面支援中文
  • 支援 Node.js + 持久化儲存
  • 免費額度每月約 $5，超過需付費
F

先不決定部署平台

先專注開發，保持標準 Express 應用結構，之後再選擇部署平台。只要架構乾淨，遷移到任何平台都很容易。
